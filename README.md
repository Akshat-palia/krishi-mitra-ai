## 🚀 Overview

Krishi Mitra AI is an AI-powered smart farming assistant designed to help farmers make better agricultural decisions using machine learning and real-time data.

It integrates crop disease detection, chatbot assistance, weather insights, and fertilizer recommendations into a single platform.

---

## ✨ Features

### 🌿 Crop Disease Detection
- Upload plant leaf images  
- Detect diseases using deep learning (PyTorch model)  

### 🤖 AI Chatbot (KissanBot)
- Multilingual assistant  
- Helps farmers with queries and guidance  

### 🌦 Weather Forecast
- Real-time weather updates  
- Helps plan irrigation and harvesting  

### 💰 Market Price Tracker
- Get latest crop prices  
- Make informed selling decisions  

### 🧪 Fertilizer Recommendation
- Suggests optimal fertilizers  
- Based on crop and soil needs
